======= ERREUR DE TYPE CARACTERE EN TROP =======
"includee" à la place de "include" -> ERROR : unknown directive "includee" in file
"." au milieu de la ligne entre "include" et ";" -> ERROR : invalid number of arguments for directive "include" in file


Pour tous mots-clés comme : "error_page", "listen", "root", "autoindex" etc -> Si caractère juste avant ou après -> Error : invalid character in directive name
                                                                            -> Sinon si espace -> OSEF -> même si 150 espaces OSEF
